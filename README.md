# Arch Gaming Setup

Script personal para preparar Arch Linux para gaming.

Incluye:
- Drivers GPU
- Steam / Lutris
- Wine + Proton GE
- Vulkan stack
- Mangohud, Gamemode, Gamescope

Uso:

git clone https://github.com/Tony-855/arch-gaming-setup.git

cd arch-gaming-setup

chmod +x gaming-setup.sh

./gaming-setup.sh
